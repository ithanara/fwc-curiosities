## 📖 Overview

This project builds a structured dataset containing the official squads for the FIFA World Cup 2026.

The data was collected from the Wikipedia page containing the tournament squads and then transformed into a clean and analysis-ready dataset using Python and Pandas.

The current stage focuses on:

* Extracting player information from the source tables
* Cleaning and standardizing fields
* Handling missing values
* Splitting combined attributes into separate columns
* Creating a consolidated dataset containing all national teams

The resulting dataset will serve as the foundation for future exploratory analyses and visualizations.

> 🚧 This project is currently a Work in Progress (WIP). Data extraction and cleaning are being finalized before the analysis phase begins.
